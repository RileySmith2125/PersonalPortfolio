Tinytype is a 4 kB library for building terminal-style web interfaces that actually align. It sets text on a strict monospace grid so columns line up the way they would in a real terminal.

It started as a dependency I kept rewriting for my own projects. Eventually it was cleaner to make it a real, tiny, dependency-free thing.
